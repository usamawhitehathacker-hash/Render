## What happens as user interacts:

```
STATE 1: Page loads (0 scroll)
━━━━━━━━━━━━━━━━━━━━━━━━━━━
┌ Announcement bar ─── rotates messages every 5s ┐
├ Header ─── cream bg, no shadow ────────────────┤
├ Hero ─── video starts playing / image visible ─┤
└────────────────────────────────────────────────┘

STATE 2: User scrolls down (50px+)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
┌ Announcement bar ─── scrolls UP and DISAPPEARS ┐
├ Header ─── STICKS to top ──────────────────────┤
│             Background → white + blur           │
│             Shadow appears (subtle)             │
├ Hero ─── scrolls normally with page ───────────┤
└────────────────────────────────────────────────┘

STATE 3: User hovers announcement bar
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Message rotation PAUSES
Close button (×) becomes more visible

STATE 4: User clicks × on announcement
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Bar slides up (height → 0)
Header moves up to fill space
Preference saved in session
Page reloads → bar stays closed

STATE 5: User hovers nav link
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Underline animates from left → right (width 0 → 100%)
Duration: 0.3s

STATE 6: User clicks hamburger (mobile)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Hamburger ☰ → animates to × (cross)
Full-screen menu slides from RIGHT
Body scroll LOCKED (no background scrolling)
```
